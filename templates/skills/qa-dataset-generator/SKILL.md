---
name: qa-dataset-generator
description: Genera datasets de prueba reutilizables para QA.
---

Objetivo

Crear conjuntos de datos que permitan ejecutar todos los casos de prueba.

Tipos de dataset

Dataset válido base.

Dataset alternativo válido.

Dataset inválido.

Dataset edge.

Dataset duplicado.

Formato

Usar ejemplos JSON o estructuras equivalentes.

Cada dataset debe incluir

Datos completos.

Descripción de uso.

Casos donde aplica.

Ejemplo

dataset-valid-product

{
  "name": "Producto Test",
  "price": 100,
  "sku": "TEST-001"
}
